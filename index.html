Introduction

Python is an easy to learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming. Python’s elegant syntax and dynamic typing, together with its interpreted nature, make it an ideal language for scripting and rapid application development in many areas on most platforms.

Hello World

  1. Open a text editor or an Integrated Development Environment (IDE) of your choice.
  2. Create a new file and save it with a .py extension, such as hello_world.py.
  3. In your file, type the following code:
    4. print("Hello, World!").

Data Types

Python has several built-in data types that are categorized into various classes:

  • Text Type: str
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool

Variables

You use variables as symbolic names for values in your application. A variable can hold a value for any data type. A variable can be called with the code X = “Hello World” From now on any time the variable X is used it will fill in the value with “Hello World”. Exe:

Print(X) #will output the string “hello world”

If Statements

An if statement can be used to run code ONLY when a curtain case is fulfilled.
Any number of elif statements can be added on to an if statement for other cases.
An else statement can be added to the end of an if statement to run code when no other cases are met

Example:

x = int(input("Please enter an integer: ")) #requests the user to input a number

if x < 0: #if number inputted is less than 0 the code prints “Negative Number”
print('Negative number')

elif x == 0: #if number inputted is equal to zero the code prints “Zero”
print('Zero')

else: #If neither case is met (ie the number is not negative and is not zero) the code prints “Positive Number”
print(Positive Number)

For Statements

Python’s for statement iterates over the items of any sequence (a list or a string), in the order that they appear in the sequence.

Example

words = ['cat', 'window', 'defenestrate'] #list with 3 different words

for w in words: #itterates once for each word in the list
print(w, len(w)) #prints the w’th word from the list with the number of characters in it

. . . output . . .
cat 3 window 6 defenestrate 12
